# 💰 Paisa – Smart Personal Finance Tracker (India)

Paisa is a modern, premium personal finance web app designed for Indian users to track expenses, income, and manage money across multiple accounts like Cash, Bank, UPI, and Credit Cards.

---

## 🚀 Live Demo

👉 https://mypaisa-nk.netlify.app

---

## ✨ Features

* 📊 Track income and expenses easily
* 💳 Multiple accounts support (Cash, Bank, UPI, Card)
* 🧾 Categorized transactions with subcategories
* 📈 Visual charts and analytics
* 📱 Mobile-first responsive design
* 🌙 Premium dark UI (Saffron theme)
* ⚡ Fast and lightweight (PWA-ready)

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Custom UI Design)
* JavaScript (Vanilla JS)
* Progressive Web App (PWA)
* Netlify (Deployment)
